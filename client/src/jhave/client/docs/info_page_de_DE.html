Willkommen bei JHAVÉ 2.0

JHAVÉ 2.0 befindet sich noch in der Betatestphase. Daher freuen wir uns über Rückmeldungen per Email zu Problemen.

Auf http://jhave.org können Sie sich über Updates informieren. Wenn Sie die Anwendung über Java WebStart tarten, werden Aktualisieren "automagisch" durchgeführt, wenn Sie den Client starten.

Wie nutzt man JHAVÉ?

Die Nutzung von JHAVÉ ist sehr einfach. Folgen Sie einfach den Anweisungen im unten stehenden Panel. Stellen Sie sicher, dass der Server und die Kategorie entsprechend Ihrer Verstanstaltung ausgewählt wurden. Sobald Sie mit dem Server verbunden sind, wählen Sie einfach einen Algorithmus aus, klicken auf Visualisieren und geben, falls erforderlich, die Eingabedaten für den Algorithmus an. Das war's schon!
