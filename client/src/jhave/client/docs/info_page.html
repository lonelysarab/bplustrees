How do I use JHAVÉ?

Using JHAVÉ is as simple as following the directions in the panel below! Make sure the server and category are correct for your class and connect to the server. Once connected select an algorithm, click visualize, and, if needed, select or create the dataset that will provide input to the algorithm. It's that easy!
