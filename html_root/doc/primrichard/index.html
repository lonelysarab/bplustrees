Prim's Algorithm

Here is pseudo code describing Prim's algorithm:


1. /* initialization */
2. PriorityQueue openList = { startVertex }
3. /* loop */
4. while ( all vertices not closed )
   {
5.    closingVertex = vertex in openList with
6.       edge of minimum weight connecting it to a closed vertex
7.       (or just the startVertex the first time through the loop);
8.    remove closingVertex from openList;

9.    for each non-closed vertex with an edge from closingVertex
10.      put vertex into openList;
   }

Note: When the inner for loop above goes through vertices connected to a node, it goes through them in alphabetical order.
