Whereas Dijkstra's shortest path algorithm finds the shortest path from a single source vertex to all other vertices, Floyd shortest path algorithm is an all-pairs shortest path algorithm since it determines the (length of) the shortest path between all pairs of vertices in a network. Here you see an adjacency matrix representation of the edges in a connected network. Floyd's algorithm keeps track of the lengths of the shortest paths between vertices in an auxiliary distance array. The distance array is initialized as follows: distance[i,j] =

  • 0 if i = j
  • edgeweight(i, j) if such an edge exists
  • UNDEFINED otherwise
