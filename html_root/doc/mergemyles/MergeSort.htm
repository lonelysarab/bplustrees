Merge Sort

Merge sort is a recursively designed algorithm. It executes in the following fashion:

  1. Return if input is a single data element.
  2. If multiple data items are in input, split in half and sort each half.
  3. Merge the sorted halves back together.

This algorithm is known as a divide-and-conquer technique. It divides the set in half and then attacks each half independantly. Within each half it repeats itself dividing the set again in half and attacking each half.. Thus, if the list to be sorted consisted of eight items, merge sort would break the set up into sets of four, then sets of two, then sets of one.
