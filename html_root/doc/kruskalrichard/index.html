Kruskal's Algorithm

Here's pseudo code describing Kruskal's algorithm:


/* initialization */
PriorityQueue = { all edges (priority determined
                  by smallest edgeweight) }
Create a UnionFind ADT with universe consisting
    of all vertices in the graph
    
/* loop */
while ( number of edges added to MST is not
        equal to number of vertices minus 1 )
{
  Remove an edge containing X and Y from
    the UnionFind ADT

  if ( not find( X, Y) )
    Add edge X-Y to UnionFind ADT
    Perform union( X, Y )
}

Note: When two edges in the priority queue have the same weight, the one with the vertex coming first in alphabetical order is removed first.
