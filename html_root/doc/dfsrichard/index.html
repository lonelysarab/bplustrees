Depth First Graph Search Algorithm

The following pseudo code describes the depth first graph search algorithm:


1. /* initialization */
2. stack openList = { startVertex }
3. /* loop */
4. while ( closedNodes != numberOfNodes && !openList.empty())
   {
5.    while ( openList.top() is a closed vertex ) openList.pop();
6.    closingVertex = openList.pop();

7.    for each non-closed vertex with an edge from closingVertex
      {
8.      openList.push( vertex );
      }
   }

Note: When the inner for loop above goes through vertices connected to a node, it goes through them in "reverse alphabetical order". Hence the vertex that comes first in alphabetical order will be placed on top of the stack.
