How do I use Gaigs?

Remember that you can use your mouse in the graphics window to drag the display of the algorithm -- particularly convenient when part of the display is off the screen or poorly centered.

Buttons!

The Gaigs plug-in for the JHAVÉ Control Panel uses the VCR-like control to allow you view the algorithm depicted in the graphics window.

  • Back - takes you to the previous slide in the slideshow. If this button is grayed out, then there is no previous slide to the one you are currently viewing.
  • Next - takes you to the next slide in the slideshow. If this button is grayed out, then this is the end, my friend.
  • Magnify - Scales up the visualization size.
  • Shrink - Scales down the visualization size.

Slider Bar

This slider allows you to move directly to any particular slide in the show.

Option Menu

If the Show Questions checkbox is checked, then questions will be displayed on the screen for you to answer. You can keep going and ignore the questions altogether if you wish, or even uncheck this box and never see the questions. Otherwise, you can try to answer the question and close the dialog box when you are done. The Connection Config option currently does not save any configuration information that you enter.

File Menu

If you are taking a course in which your instructor is using JHAVÉ to administer "for-real" quizzes, you will use the quiz mode box to indicate that you are ready to take a quiz.
