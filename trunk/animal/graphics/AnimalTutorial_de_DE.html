Wie man Animal nutzt

  1. Laden Sie eine Animation auf eine der folgenden Arten:
    • Datei -> Öffnen
    • Datei -> Importieren
    • Datei -> Datenbank
    • Direkte Eingabe von AnimalScript-Code mittels Datei -> AnimalScript Code
    • Alternativ können Sie die entsprechenden Buttons der Toolleiste nutzen (der zweite Block der Leiste)
  2. Das Animationsfenster öffnet sich automatisch. Hier finden Sie die Hauptkontrollen:
    1. links oben können Sie die Ablaufgeschwindigkeit einstellen (0-1000%)
    2. rechts oben können Sie den Vergrößerungsfaktor einstellen (0-500%)
    3. links unten ist die Schaltleiste mit den folgenden Knöpfen:
      1. Sprung an den Anfang
      2.  Einen Schritt zurückspringen
      3. Animation schrittweise rückwärts ausführen (mehrere Schritte)
      4.  Aktuellen Schritt rückwärts ausführen
      5.  Pause
      6. Aktuellen Schritt vorwärts ausführen
      7.  Animation schrittweise vorwärts ausführen (mehrere Schritte)
      8.  Einen Schritt vorspringen
      9.  Sprung an das Animationsende
    • In der Mitte unten können Sie direkt einen Zielschritt angeben.
    • Unten rechts können Sie sehen, wieviel Prozent der Animation schon angezeigt wurden. Sie können hier den Regler auch ziehen, um schnell voranzukommen.
