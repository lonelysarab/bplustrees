How to Use Animal

  1. Load an animation in one of the following ways:
    • File -> Open
    • File -> Import...
    • File -> Repository
    • Direct input of AnimalScript-Code via File -> AnimalScript Code
    • Alternatively, you can use the corresponding buttons in the toolbar (the second block of the toolbar).
  2. The animation window will open automatically. Here you will find the main controls:
    1. you can adjust the animation speed using the slider at the top left (0-1000%)
    2. magnification can be adjusted between 0-500% using the slider at the top right
    3. at the bottom left, you will find the main control buttons:
      1.  Jump to the start of the animation
      2. jump back one step
      3.  reverse slide show
      4. play current step in reverse mode
      5. pause
      6. play current stepktuellen Schritt vorwärts ausführen
      7. slide show
      8. jump one step forward
      9. jump to the end of the animation
    • In the bottom center, you can directly enter the target step number
    • The slider at the bottom right shows how much of the animation was already shown. You can also drag the slider for a "fast forward / fast reverse".
