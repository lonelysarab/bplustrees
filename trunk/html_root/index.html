Test Site for Reattore

This directory contains a set of pages to test Reatore and other servers against. The files are:

  • Text files:
    • small.xml (275 bytes)
    • medium.html (19120 bytes)
  • Binary files:
    • logo.png (3771 bytes)
    • medium.bin (32768 bytes)

Other parts of this site:

  • HTTP pipelining test page
  • A missing link to show fetching the referer header.

The following features are available when using the max-config.xml configuration:

  • Reattore project documentation, showing partitioning ('aliases')
  • Server statistics, showing a simple servlet.
  • A 'naked' version of this site, showing auto indexing.
