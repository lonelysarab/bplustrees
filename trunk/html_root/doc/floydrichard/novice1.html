Floyd's Algorithm (Novice)

Floyd's Algorithm calculates the minimal path distances between all pairs of vertices in a directed graph. The alogorithm uses a two dimensional array to represent these values, initially set up for each cell as follows:

IfValue
1st vertex equals 2nd vertex (path to itself)0
No edge from 1st vertex to 2nd vertexBig number
Edge exists from 1st vertex to 2nd vertexEdge weight
