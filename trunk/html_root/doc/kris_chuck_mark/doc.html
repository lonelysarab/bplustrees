Infix-to-postfix

Converting an infix expression to postfix is an application using queues and stacks. Students should already understand the details of the queue and stack ADTs. Queue operations used are enqueue(), dequeue(), and isEmpty(). Stack operations used are pop(), push(), peek(), and is Empty().

Order of precedence must be preserved during the conversion. Each token of the infix expression will be dequeued; its token type will be determined; and the token, along with other relevant tokens, will be handled properly depending on the token type.

The token type will be denoted by colors:

  • operand Greenoperations Blue
  • ( Yellow
  • ) Red
Also, the empty infix queue will be denoted by purple. Each snapshot will illustrate the handling of one token.
