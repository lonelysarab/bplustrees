Dijkstra's Algorithm

Here is pseudo code describing Dijkstra's algorithm:


1.  /* initialization */
2.  PriorityQueue openList = { startVertex }
3.  array cost = { Big, Big, Big, ... }
4.  array pred = { No, No, No, ... }
5.  cost[startVertex] = 0;

6.  /* loop */
7.  while ( !openList.empty() )
    {
8.    closingVertex = vertex in openList with minimum cost;
9.    remove closingVertex from openList;

10.   for each non-closed vertex with an edge from closingVertex
      {
11.     if (cost[vertex] > cost[closingVertex] + edgeWeight)
        {
12.        cost[vertex]
13.          = cost[closingVertex] + edgeWeight;
14.        pred[vertex] = closingVertex;
15.        put vertex into openList;
        }
      }

    }

Note: When the inner for loop above goes through vertices connected to a node, it goes through them in alphabetical order.
