Beta 2

New to Beta 2:
  • Client now uses the native look and feel of the host operating system. This makes it look much nicer under Windows and feel better under Linux.
  • Gaigs has been reworked to look nicer under Windows and with most native look and feels.
  • Revised play control icons for Animal to make it look better under native look and feels.
  • Installer has been upgraded to the latest version of InstallAnywhere Now!
Fixed in Beta 2:
  • If a True / False questions answer is False it now works correctly.

Beta 1

New to Beta 1:
  • The client no longer connects to the server on opening but instead prompts the user for a category and server then connects.
  • The opening the info page has been replaced with instructions on how to use JHAVÉ.
  • The installer now displays this revision history during install.
  • This history information is also available from the about box.
  • The about box has been redesigned, it now includes a small amount of information about the author and contributors to the client.
Fixed in Beta 1:
  • Questions with long questions are now displayed correctly.

Alpha 3

New to Alpha 3:
  • Renamed from prototype 3 to Alpha 3 since it implements most of the features.
  • Connection History has been moved to the debug tab.
  • Algorithms with InputGenerators should now work, also you can choose from new or old data now!
  • User can now login to the server to take tests.
  • The question system is now fully implemented!
  • Installer has been upgraded from InstallAnywhere Now! 4.5 to InstallAnywhere Now! 5.
  • There is now a command line option for: debug mode, selecting a category, and printing a help message(--help).
  • A lot of major and minor changes.
Fixed in Alpha 3:
  • A lot of major and minor bug fixes.
Missing from Alpha 3:
  • A graphical way to select the category before connecting to the server.
  • Probably something, especially testing on taking a quiz.

Prototype 2

New to Prototype 2
  • Connection to the server is made at start up. Visualizations without input generators can be viewed.
  • A new Connection tab that displays to the user information about client / server communications.
  • A new connection status bar that displays the last transaction made between client and server.
  • A new debug tab, it is not implemented yet but will display System.out and System.err messages.
Fixed in Prototype 2
  • Changes made to the Connection configuration can now be saved.
Missing from Prototype 2
  • The question system is completly missing from this release.
  • The system for taking quizes for a grade is completly missing as well.
  • Visualizations with input generators cannot be viewed.

Prototype 1

  • Design prototype, no functionality is implemented as it is just a mock up of what the interface will look like.
